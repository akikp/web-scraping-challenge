# Mission to Mars

Flask web application that scrapes various websites for data related to the Mars Mission and displays the information in a single HTML page.

# Requirements

* Modules Required
    - pandas
    - splinter
    - pymongo
    - Flask
    - requests
    - beautifulsoup4
    - Flask-PyMongo

# Scrapping

* NASA Mars News
* JPL Mars Space Images - Featured Image
* Mars Facts
* Mars Hemispheres
